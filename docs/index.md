# Site Python

Это учебный статический сайт на **MkDocs** с собственной темой на HTML, CSS и JS.

## Что здесь реализовано

- Кастомные **header** и **footer** (шаблоны Jinja2)
- Стилизованная главная страница (отдельный шаблон `home.html`)
- Сборка стилей через **PostCSS** (автопрефиксы + минификация)
- Минификация JS и HTML
- Валидация HTML на этапе CI
- Деплой на **GitHub Pages** через GitHub Actions

Перейдите в репозиторий, чтобы посмотреть отчет по сборке и деплою.
